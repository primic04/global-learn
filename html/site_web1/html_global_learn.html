Aide mémoire de tous ce que j'ai appris

résumé du chapitre 1

  • le web a été inventé par Tim berner-lee au debut des année 1990
  • Pour créer des sites web, on utilise deux type de langage informatique

    1. html pour écrire et organiser le contenue de sa page(paragraphe, titre etc...)
    2. css pour mettre en forme la page(couleur, taille, font-size)

  • il y a eu plusieurs version des langage html et css dont les derniers venu sont html 5 et css 3
  • le navigateur web est un programme qui sert à afficher des sites web. Il lit le code HTML5 et CSS3 pour savoir ce qu'il doit afficher
  • il existe de nombreux navigateur web different: chrome, firefor, internet explorer, safari, opéra...

résumé du chapitre 2

  • on utilise l'éditeur de texte pour créé un fichier ayant comme extention .html(par exemple: test1.html.)ce sera notre page web
  • ce fichier s'ouvre dans le navigateur, simplement en double-cliquant sur son nom
  • à l'interieur de notre fichier, nous écrirons le contenu de notre page, accompagné de balise HTML
  • les balises peuvent prendre plusieurs forme:

    1. les balises en paires, elles s'ouvrent et se ferment pour delimiter le contenu(debut et fin d'un titre,par exemple)
    2. les balises orphelines(on ne les insères qu'en un seul exemplaire). elles servent à inserer un élément à un endroit précis(par exemple une image)

  • les balises sont parfois accompagné d'attribut pour donné des indications supplémmentaires(exemple: image nom"photo.jpg")
  • une page web est constituée de deux sections principales: l'en-tête(head) et le corps(body)
  • on peut afficher le code source de n'importe quelle page web en cliquant-droit sur la page et en séléctionnant (inspecter)(inspect)

résumé du chapitre 3

  • le HTML comporte de nombreuses balises qui servent à organiser le texte de notre page. elles donnent des indications comme << ceci est un paragraphe >> << ceci est un titre >> etc...
  • les paragraphe sont défini avec la balise << p >> et les saut a la ligne par la balise << br >>.
  • il existe six niveaux de titre, de (h1) a (h6), à utilisé selon le degré d'importance du titre. il apparaitra plus gros selon le degré d'importance du titre
  • on peut mettre en valeur certains mot avec les balises (strong)-(em) et (mark)
  • pour créer des listes, on doit utiliser des balise (ul (unorder list))-(ol (order list)). À l'interieur, on y insère les éléments avec une balise (li (liste)) pour chaque item

résumé du chapitre 4

  • les liens permettent de changer de page et sont, par default, écrit en bleu soulignés
  • pour insérer un lien, on utilise la balise (a) avec l'attribut (href) pour indiquer l'adressse de la page URL ciblé

    exemple: < < a href ="http://www.google.com"> >

  • on créé un lien vers une autre page de son site simplement en écrivant le nom du fichier

    exemple: < < a href ="page2.html"> >

  • les liens permettent aussi d'amener vers d'autre endroit sur la même page (on l'appelle aussi #ancre). on le créé avec l'attribut (id) pour marquer un endroit dans la page, puis ajouter un lien vers l'ancre:

    exemple: < < a href ="#ancre" > >

résumé du chapitre 5

  • il existe plusieurs formats d'image adaptés au web

    1. jpeg: pour les photos
    2. png: pour les autres illustrations
    3. gif: similaire au png, plus limité en nombre de couleurs mais qui peut être animé

  • on insère une image avec la balise < < img src="" alt=""> >, qui dois obligatoirement comporter deux types d'éléments (src(nom de l'image))-(alt(courte description de l'image))
  • si une image illustre le texte (et elle n'est pas seulement décorative) il est conseillé de la placer au sein d'une balise (figure). la balise < < figcaption > > sert à lui associer une légende.
